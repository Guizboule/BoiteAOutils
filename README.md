# BoiteAOutils

La boîte à outils est un application Android dans laquelle on trouve une multitude d'outils disponible pour l'utilisateur. 


## Les outils disponibles

* Boussole
* Lampe torche

## Les outils à venir

* Analyseur de son
* Niveau à bulle
